📦 STEP 1 — Pehle Ledger Banayein (sirf ek baar)

Yeh chote XML se aapke ledger names (Bank, Suspense, Rule wale ledgers) Tally mein automatically create ho jayenge. Agar yeh ledger already Tally mein bane hain, to yeh step skip kar sakte hain.

📊 STEP 2 — Saari Entries Import Karein

Step 1 import karne ke baad yeh file import karein — rules wali entries seedha sahi ledger mein, baaki Suspense mein jaayengi.

📥 Tally mein import karne ka tarika (dono files step-by-step):

  1. Tally Prime kholen → apni company open karein
  2. Gateway of Tally → Import Data → Masters → Ledger Master XML file select karein (Step 1 wali)
  3. Phir Gateway of Tally → Import Data → Vouchers → Voucher XML file select karein (Step 2 wali)
  4. Enter dabayein → saari entries bina exception ke import ho jayengi!
